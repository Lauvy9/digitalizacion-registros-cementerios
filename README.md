#  Sistema de Digitalización de Registros Cementerialos

##  Descripción del Proyecto
Sistema integral para la digitalización y gestión automatizada de registros de cementerios, desarrollado como proyecto final para la Tecnicatura en Ciencia de Datos e Inteligencia Artificial.

###  Objetivos
- Digitalización de tumbas antiguas sin registro
- Automatización de nuevos registros de fallecimientos  
- Georreferenciación precisa de cada tumba
- Dashboard interactivo para gestión visual

###  Características Principales
-  Generación automática de IDs únicos
-  Integración con QGIS para mapeo
-  Formularios digitales de captura
-  Validaciones automáticas de datos
-  Dashboard interactivo en Streamlit
